# Hacker / Cyberpunk Design Direction

## Core Identity

The platform should feel like:
- a futuristic underground marketplace,
- an elite digital trading hub,
- a cyberpunk collector ecosystem,
- a secure encrypted network for rare digital assets.

The experience should combine:
- hacker aesthetics,
- premium gaming UI,
- terminal-inspired interactions,
- futuristic dashboards,
- neon cyberpunk visuals.

Inspired by:
- Blade Runner
- Cyberpunk 2077
- Mr. Robot
- Deus Ex
- futuristic darknet UI aesthetics
- sci-fi command terminals

---

# Visual Style

## Atmosphere

The interface should feel:
- dark,
- encrypted,
- intelligent,
- immersive,
- high-tech,
- slightly mysterious,
- futuristic but clean.

Avoid:
- cartoonish hacker themes,
- overly noisy glitch effects,
- low-quality neon spam,
- unreadable interfaces.

---

# Color Palette

Primary Background:
- #050505
- #0a0a0a
- #111111

Accent Colors:
- Neon Cyan
- Electric Purple
- Matrix Green
- Deep Blue

Optional Highlights:
- Red alerts
- Amber warnings

Use gradients sparingly.

---

# UI Effects

Allowed effects:
- subtle scanlines
- glassmorphism
- soft glow borders
- animated gradients
- terminal cursors
- holographic cards
- hover glows
- data stream animations
- grid overlays

Avoid:
- excessive motion,
- flashing animations,
- distracting glitches.

---

# Typography

Primary:
- Inter

Secondary:
- JetBrains Mono
- Space Mono
- Orbitron

Use monospace fonts for:
- wallet addresses,
- terminal panels,
- transaction logs,
- IDs,
- rarity metadata.

---

# Component Style

Cards should look:
- holographic,
- encrypted,
- premium,
- collectible.

Buttons should:
- glow softly on hover,
- feel tactile,
- have futuristic transitions.

Panels should resemble:
- command consoles,
- encrypted dashboards,
- AI control systems.

---

# Animation Philosophy

Animations should feel:
- smooth,
- stealthy,
- intelligent.

Examples:
- subtle hover glows,
- data loading streams,
- animated scan lines,
- floating particles,
- digital flickers,
- smooth panel reveals.

Never use:
- chaotic transitions,
- excessive bouncing,
- childish motion effects.

---

# Marketplace Atmosphere

The marketplace should feel like:
- trading rare classified digital artifacts,
- accessing hidden elite collections,
- operating inside a secure cyber network.

Product drops should feel:
- exclusive,
- time-sensitive,
- rare.

---
